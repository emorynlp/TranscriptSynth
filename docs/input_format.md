# Input Format

## Transcript with Speaker Diarization

```json
[
    {
        "sid": "1",
        "utt": "Hello",
        "dur": [0, 9.2]
    },
    {
        "sid": "2",
        "utt": "How are you",
        "dur": [10.00, 14.23]
    }
]
```

## Transcript without Speaker Diarization

```json
[
    {
        "utt": "Hello",
        "dur": [0, 9.2]
    },
    {
        "utt": "How are you",
        "dur": [10.00, 14.23]
    }
]
```
