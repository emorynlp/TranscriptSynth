# Input Format

## Transcript with Speaker Diarization

```json
[
    {
        "sid": "1",
        "text": "Hello",
        "time": [0, 9.2]
    },
    {
        "sid": "2",
        "text": "How are you",
        "time": [10.00, 14.23]
    }
]
```

## Transcript without Speaker Diarization

```json
[
    {
        "text": "Hello",
        "time": [0, 9.2]
    },
    {
        "text": "How are you",
        "time": [10.00, 14.23]
    }
]
```
